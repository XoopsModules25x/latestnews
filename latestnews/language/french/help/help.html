Help: LatestNews

Description


This module displays the latest news from the News module (version 1.70+) in a block.
.

Features:

  • Can set the number of articles to display, using one or more columns.
  • Can choose to display the title, author, date, comments link, and other display settings.
  • Can customize image properties in all articles (i.e. width, height, position, borders).
  • Can customize the number of letters to truncate each news item, without cutting off any words midway.
  • Adding link and description to each image automatically for SEO.
  • Can limit by topic.
  • Can display featured news only.
  • Can clone unlimited blocks.
  • Can scroll the articles.
